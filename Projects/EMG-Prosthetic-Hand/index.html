EMG-Controlled Prosthetic Hand

(In Progress)

Designing a tendon-driven prosthetic hand that will ultimately use forearm EMG signals to control grasping motions in real time.

Project Overview

This project focuses on designing and prototyping a low-cost, tendon-driven prosthetic hand using 3D printed parts, embedded electronics, and signal processing. The long-term goal is to convert forearm EMG signals into coordinated finger motion for intuitive prosthetic control.

Design Objectives

  • Affordable 3D-printed prosthetic hand
  • Tendon-driven finger actuation
  • Arduino-based control electronics
  • EMG signal integration
  • Coordinated grasping motion

System Architecture

The current prototype validates the mechanical actuation system before integrating EMG signal processing. Four independently actuated fingers are driven by SG90 servo motors through tendon routing while an Arduino Uno provides synchronized control. Future iterations will replace pre-programmed servo commands with real-time forearm EMG input.

Mechanical Design

The prosthetic hand was designed entirely in SolidWorks with an emphasis on modularity, low manufacturing cost, and tendon-driven actuation. Each finger underwent multiple design iterations to improve joint motion, tendon routing, and assembly before being integrated into the complete four-finger prototype.

Design Philosophy

This prototype focuses on achieving reliable grasping through a simplified tendon-driven mechanism. Each finger is independently actuated by a servo motor while remaining modular so components can be redesigned without rebuilding the entire hand.

Actuation

Fishing line tendons driven by SG90 servos generate finger flexion.

Extension

Elastic bands provide passive finger extension once tendon tension is released.

Construction

Modular PLA components assembled using M3 fasteners and printed on an FDM 3D printer.

Manufacturing

Designed entirely in SolidWorks with iterative CAD and prototype testing.

Electronics & Control

The prototype is controlled using an Arduino Uno that independently commands four SG90 micro servos through PWM outputs. An external regulated 5V supply powers the servos while sharing a common ground with the Arduino to ensure stable operation.
