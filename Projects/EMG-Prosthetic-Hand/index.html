EMG-Controlled Prosthetic Hand

(In Progress)

Designing a tendon-driven prosthetic hand that will ultimately use forearm EMG signals to control grasping motions in real time. The current prototype validates the mechanical design using Arduino-controlled servo motors prior to EMG integration.

Project Overview

This project focuses on designing and prototyping a low-cost, tendon-driven prosthetic hand using additive manufacturing, embedded electronics, and biomedical engineering principles. The long-term goal is to convert forearm EMG signals into coordinated finger motion for intuitive prosthetic control.

Design Objectives

  • Affordable 3D-printed prosthetic hand
  • Tendon-driven finger actuation
  • Arduino-based control electronics
  • EMG signal integration
  • Coordinated grasping motion
  • Modular design

System Architecture

The current prototype validates the mechanical actuation system before integrating EMG signal processing. Four independently driven fingers are actuated by SG90 servos through tendon routing, with Arduino Uno providing synchronized control. Future iterations will replace manual commands with real-time forearm EMG input.

Mechanical Design

The prosthetic hand was designed from the ground up in SolidWorks with an emphasis on modularity, simple manufacturing, and tendon-driven actuation. Each finger was iteratively redesigned through multiple CAD and 3D printing cycles to improve joint motion, tendon routing, and overall reliability.

Design Philosophy

Rather than replicating the complexity of the human hand, this prototype simplifies the mechanical system by coupling finger motion while preserving smooth tendon-driven flexion. The modular architecture allows each finger to be redesigned independently without requiring a complete rebuild.

Actuation

Fishing line tendons driven by SG90 servos generate finger flexion.

Extension

Elastic bands provide passive finger extension after tendon release.

Construction

Modular PLA components assembled using M3 fasteners.

Manufacturing

Designed entirely in SolidWorks and fabricated using FDM 3D printing.
